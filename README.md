# Conjurer

## Developing

We manage dependencies with `yarn`.

```bash
# install dependencies
yarn

# run the app with hot reloading on save
yarn dev
```

## Concepts

- timeline
- pattern

  - uniforms definition (generic + )
- effect
- block

- uniforms
